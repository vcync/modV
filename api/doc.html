# Layer#clearing

Indicates whether the Layer should clear before redraw

Type: Boolean

# Layer#alpha

The level of opacity between 0 and 1 the Layer should be muxed at

Type: Number

# Layer#enabled

Indicates whether the Layer should be drawn

Type: Boolean

# Layer#inherit

Indicates whether the Layer should inherit from another Layer at redraw

Type: Boolean

# Layer#inheritFrom

The target Layer to inherit from, -1 being the previous Layer in modV#layers, 0-n being the index of another Layer within modV#layers

Type: Number

# Layer#pipeline

Indicates whether the Layer should render using pipeline at redraw

Type: Boolean

# Layer#drawToOutput

Indicates whether the Layer should mux to the output canvas on the output window

Type: Boolean

# Layer#locked

Indicates whether the Layer is Locked

Type: Boolean

# Layer#blending

The Blendmode the Layer muxes with

Type: String

# mux

Clears the modV Output Canvas, then loops over all Layers' Canvases to draw them all onto the output Canvas.

Returns Promise Resolves when all Layers have been muxed to the output Canvas.

# render2d

Renders a frame for 2D Module type

# Parameters

  • args Object Arguments
    • args.Module Object A 2D Module
    • args.canvas HTMLCanvasElement The Canvas to draw to
    • args.context CanvasRenderingContext2D The Context of the Canvas
    • args.video HTMLVideoElement The video stream requested by modV
    • args.meyda Meyda The Meyda instance (for Windowing functions etc.)
    • args.delta DOMHighResTimeStamp Timestamp returned by requestAnimationFrame
    • args.bpm Number The detected or tapped BPM
    • args.kick Boolean Indicates if BeatDetektor detected a kick in the audio stream
    • args.meydaFeatures Array<MeydaFeatures> Requested Meyda features
    • args.features

# renderIsf

# Parameters

  • args Object Arguments
    • args.Module ModuleISF An ISF Module
    • args.canvas HTMLCanvasElement The Canvas to draw to
    • args.context CanvasRenderingContext2D The Context of the Canvas
    • args.pipeline boolean Should we clear the Canvas before drawing?
